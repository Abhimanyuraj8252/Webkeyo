## 🌟 Overview

**Webkeyo** is a cutting-edge Flutter application designed to bridge the gap between static content and dynamic video storytelling. Whether you're a manga enthusiast, a student, or a content creator, Webkeyo empowers you to generate professional-grade script and video summaries using advanced Vision AI and TTS technologies.

## ✨ Key Features

- **🖼️ Multi-Source Image Extraction**: Seamlessly extract images from `.cbz`, `.zip`, `.pdf`, or local directories.
- **👁️ Vision AI Analysis**: Industry-leading integration with **Google Gemini**, **OpenRouter**, **Anthropic**, **OpenAI**, and now **Groq Vision** for precise content understanding.
- **✍️ AI Scripting & Editing**: Automatically generate scripts in multiple languages (Hindi, Hinglish, English, etc.) with a built-in professional editor.
- **🎙️ Advanced TTS Integration**: High-quality audio generation using **Microsoft Edge TTS**, **ElevenLabs**, **Deepgram**, and more.
- **🎬 Cinematic Video Rendering**: Powerful FFmpeg-based rendering with custom resolution support (up to 4K) and YouTube-fit cropping.
- **🔄 Flexible Pipeline**: Navigate between project stages (Images → Script → Audio → Video) with a modern, reactive UI.

## 🛠️ Technology Stack

- **Framework**: [Flutter](https://flutter.dev/) - For a smooth, native cross-platform experience.
- **State Management**: [Provider](https://pub.dev/packages/provider) - Robust and scalable architecture.
- **Local Database**: [Hive](https://hivedb.dev/) - Lightning-fast NoSQL storage for projects and AI providers.
- **Engine**: [FFmpeg](https://ffmpeg.org/) - The industry standard for video and audio processing.
- **Animations**: [Flutter Animate](https://pub.dev/packages/flutter_animate) - Premium micro-interactions and transitions.

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (latest version recommended)
- Android Studio / VS Code
- FFmpeg installed on the device/environment

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Abhimanyuraj8252/Webkeyo.git
   cd Webkeyo
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the application**
   ```bash
   flutter run
   ```

## ⚙️ AI Configuration

Webkeyo supports a wide range of AI providers. To get started:
1. Navigate to **Settings** → **AI Providers**.
2. Enable your preferred providers (e.g., Groq, Gemini, OpenAI).
3. Enter your API keys and you're ready to go!
